# {NUMBER\_ALPHA}

## Example

```
🐨・{NUMBER_ALPHA} Voice Channel
```

How multiple temporary channels from a creator could look like:

```
🐨・A Voice Channel
🐨・B Voice Channel
🐨・C Voice Channel
🐨・D Voice Channel
🐨・E Voice Channel
```
