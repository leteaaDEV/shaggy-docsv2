# {NUMBER\_ROMAN}

## Example

```
🦊・{NUMBER_ROMAN} Voice Channel
```

How multiple temporary channels from a creator could look like:

```
🦊・I Voice Channel
🦊・II Voice Channel
🦊・III Voice Channel
🦊・IV Voice Channel
🦊・V Voice Channel
```
